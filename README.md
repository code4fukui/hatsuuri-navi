# hatsuuri-navi

A web application to navigate year-end and New Year's store hours for businesses in Fukui, Japan.

## Demo

- **[Main View (with filtering)](https://code4fukui.github.io/hatsuuri-navi/)**: An interactive map and table with checkboxes to filter stores by operating days.
- **[Simple View (unfiltered map)](https://code4fukui.github.io/hatsuuri-navi/map.html)**: A basic map displaying all store hours without filtering options.

## Features

- Displays store operating hours for the year-end and New Year period in Fukui.
- Provides an interactive map and table view of store locations.
- Allows users to filter the list to show only stores open on selected dates.

## Data Source

This project consumes store schedule data from a CSV file hosted by the [misakiotb/shop-schedule-fukui-data](https://github.com/misakiotb/shop-schedule-fukui-data/) repository.
