# hatsuuri-navi

福井県内の店舗の年末年始の営業時間を案内するWebアプリケーションです。

## デモ

- **[メインビュー（絞り込み機能付き）](https://code4fukui.github.io/hatsuuri-navi/)**: 営業日で店舗を絞り込むためのチェックボックスを備えた、インタラクティブな地図とテーブルです。
- **[シンプルビュー（絞り込みなしの地図）](https://code4fukui.github.io/hatsuuri-navi/map.html)**: 絞り込み機能を持たず、すべての店舗の営業時間を表示するシンプルな地図です。

## 機能

- 福井県内の年末年始の店舗営業時間を表示します。
- 店舗の場所をインタラクティブな地図とテーブルビューで提供します。
- 選択した日付に営業している店舗のみを表示するように一覧を絞り込むことができます。

## データソース

本プロジェクトは、[misakiotb/shop-schedule-fukui-data](https://github.com/misakiotb/shop-schedule-fukui-data/) リポジトリで公開されているCSVファイルから店舗の営業時間データを取得して利用しています。
